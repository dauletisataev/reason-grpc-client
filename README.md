# 1. Watch

```
yarn watch
```

# 2. Build
> each time code should be bundled to be able to send requests.
```
yarn build
```

# 3. Start
```
yarn start
```

## Generate protocols from `src/protoecho.proto` to `src/proto/`
```
yarn gen:protocol
```
> make sure to have `ocaml-protoc-plugin` installed
